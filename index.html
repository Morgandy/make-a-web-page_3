Don't Repeat yourself (DRY)

In programming princples DRY is one of the most fundamental.
Everyline of code that goes into an application must be maintained, and potential souce of future bugs.
Many process in software development are repetitive and easily automated.
Understanding the DRY princple the developer (me) will learn to recognize diplication, and understands how to eliminate it through appropriate practice and proper abstraction, can produce much cleaner code than one who continuously fills the application with unnecessary repetition.

Basics of CSS

CSS is a style sheet language used for describing the look and
